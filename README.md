# Restaurant Market Research & Data Analysis Project

## Overview
This project analyzes restaurant data collected from multiple restaurants to identify trends in ratings, pricing, cuisines, and customer preferences.

## Objectives
- Collect restaurant data
- Clean and structure datasets
- Analyze restaurant performance using SQL
- Generate business insights and recommendations

## Dataset
The dataset includes:
- Restaurant Name
- Cuisine
- Rating
- Reviews
- Cost for Two
- Delivery Time
- Restaurant Type

## Tools Used
- Excel
- Google Sheets
- SQL
- Browser DevTools

## Key Insights
- Identified top-rated restaurants.
- Analyzed cuisine popularity and pricing trends.
- Compared cloud kitchens and dine-in restaurants.
- Generated recommendations based on customer preferences.
